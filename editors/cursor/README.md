# [Cursor Editor](https://cursor.com/docs)

Sets up the Cursor configuration.

This folder contains only the equivalent config to VSCode. It should be synced in the `~/Library/Application\ Support/Cursor/`.

Cursor has other features that are configured in other places:
- `permissions.json`
- `hooks.json`
- ...

We centralize that in the [`./harnesses/cursor-cli/`](./harnesses/cursor-cli/README.md).
